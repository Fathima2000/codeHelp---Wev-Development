Selector

1. The descendant (space) selector matches all elements that are descendants of a specified element.

2. The child selector (>) selects all elements that are the children of a specified element.

3. The adjacent + selector is used to select an element that is directly after another specific element.

4. The general sibling selector (~) selects all elements that are next siblings of a specified element.
